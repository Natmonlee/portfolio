Cocktail API

Base URL: http://13.41.54.243

This API allows the user to access a database of cocktail recipes. You can filter results by cocktail name, ingredients, glass type and whether or not the cocktail contains alcohol.

Endpoints

GET /v1/cocktail-recipes get cocktail recipes from database

Parameters

Optional query strings: name, alcoholic, glass, ingredients, minIngredients, maxIngredients (all case-insensitive)

Name

Description

name

string


alcoholic

Filter by whether cocktail contains alcohol

boolean



glass

string


ingredients

comma-separated string


minIngredients(default 1)

number

maxIngredients(default infinity)

number

Response Status Code & Text

200
OK

Response Body

JSON object containing all cocktails filtered by input parameters

GET /v1/cocktail-glasses get all cocktail glasses in database

Parameters

None

Response Status Code & Text

200
OK

Response Body

Array of all glasses in database

GET /v1/cocktail-ingredients get all cocktail ingredients in database

Parameters

None

Response Status Code & Text

200
OK

Response Body

Array of all ingredients in database

POST /v1/add-cocktail add cocktail to database

Parameters

Object(JSON data format) in request body with properties:

name string required
alcoholic boolean required
ingredients object required
method string required
glass string optional
photoUrl string optional(URL must be valid)
